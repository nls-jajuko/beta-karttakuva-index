Kiinteistötietojen kyselypalvelu (OGC API Features)

Karttakuvapalvelu (Vektoritiilet) BETA
- palvelusta saa kiinteistöjaotuksen tietoja karttatiilinä.
  • Palvelun TileJSON -kuvaus, jonka voi liittää tyylitiedostoon
  • Vektorikarttatyylitiedoston esimerkki
  •  Vector Tiles -standardi, johon palvelun toteutus perustuu
